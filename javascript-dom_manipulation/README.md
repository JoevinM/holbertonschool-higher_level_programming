## Learning Objectives

At the end of this project, you should be able to explain to anyone, without the help of Google:

### General
- How to select HTML elements in JavaScript
- The differences between ID, class, and tag name selectors
- How to modify an HTML element’s style
- How to get and update the content of an HTML element
- How to dynamically modify the DOM structure
- How to make HTTP requests using **XMLHttpRequest**
- How to make HTTP requests using the **Fetch API**
- How to listen to and handle DOM events
- How to bind functions to user interactions

## Requirements

### General
- Allowed editors: **All of them**
- All scripts will be interpreted in the **Chrome browser (version 57.0 or later)**
- All files should end with a **new line**
- A mandatory `README.md` file with meaningful information must be placed at the **root of the project**
- Your code should be **semistandard** compliant
- You are **not allowed to use `var`**
- The HTML page **must not reload** for each action — all updates should be done via **DOM manipulation or Fetch API**
